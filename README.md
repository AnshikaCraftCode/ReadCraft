## What is Readcraft?

Readcraft is a free, browser-based AI README generator for developers.

You describe your project, pick your sections, choose your AI provider — and get a clean, professional `README.md` ready to drop straight into your GitHub repo.

No login. No subscription. No data stored. Ever.

---

## ✨ Features

- 🤖 **Multi-provider AI** — Works with Anthropic Claude, OpenAI GPT, Google Gemini, and OpenRouter
- ⚡ **Real-time streaming** — Watch your README write itself live
- 📄 **12 sections** — Badges, Installation, Usage, API Docs, Configuration, Roadmap, FAQ, and more
- 🎨 **4 writing tones** — Professional, Friendly, Minimal, or Fun
- 👁 **Live Preview** — Switch between raw Markdown and rendered preview instantly
- 📋 **One-click Copy** — Copy the raw Markdown instantly
- 💾 **Download as .md** — Save directly to your machine
- 🔒 **Privacy first** — Your API key never leaves your browser
- 📱 **Mobile friendly** — Works on any device, any browser

## 🔑 Getting an API Key

You need an API key from one of these providers:

| Provider | Free Tier | Link |
|---|---|---|
| OpenRouter | ✅ Yes | [openrouter.ai/keys](https://openrouter.ai/keys) |
| Anthropic | ✅ Yes | [console.anthropic.com](https://console.anthropic.com) |
| OpenAI | ✅ Yes | [platform.openai.com](https://platform.openai.com/api-keys) |
| Google Gemini | ✅ Yes | [aistudio.google.com](https://aistudio.google.com/app/apikey) |

> **Tip:** OpenRouter is the easiest option — one key gives you access to all models including Claude, GPT-4o, Gemini, Llama, Mistral and more.

## 📖 How to Use

1. **Open the app** in your browser
2. **Pick your AI provider** from the tab selector at the top
3. **Paste your API key** into the key field
4. **Fill in your project details** — name, description, tech stack, features
5. **Choose your sections** using the checkboxes
6. **Hit Generate README →** and watch it stream in live
7. **Copy or Download** your README and paste it into GitHub
